üîÑ State in Angular vs React

Angular: this.count = 0; this.count++; (mutable)

React: const [count, setCount] = useState(0); setCount(count + 1); (immutable)

Why? React needs immutability to detect changes efficiently

üîÑ Services vs Custom Hooks

Angular Service: Singleton, injected via DI, shared state

React Custom Hook: Function that returns state/functions, called in component, isolated state per component

Key Difference: Hooks create isolated instances; Angular services are singletons
